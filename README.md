Adversarial Robustness: I-FGSM Attacks and Diffusion-Based PurificationProject SummaryThis project evaluates the security of Deep Neural Networks (DNNs) against gradient-based adversarial perturbations. By implementing the Iterative Fast Gradient Sign Method (I-FGSM), the research demonstrates how minimal, targeted noise can bypass state-of-the-art classification models. To mitigate these risks, a Stochastic Purification pipeline was developed, utilizing an adaptive denoising process to restore model accuracy without requiring architectural retraining.Technical Implementation1. Adversarial Attack MethodologyThe primary threat model utilized is the Iterative Fast Gradient Sign Method (I-FGSM). Unlike single-step attacks, I-FGSM applies perturbations iteratively to ensure the adversarial example remains within a defined epsilon-ball while maximizing the loss function.Technical Specifications:Optimization: Directed gradient descent on input space.Architectures Tested: ResNet50 (ImageNet-1K) and custom Convolutional Neural Networks (MNIST).Constraints: Epsilon clipping and sign-based normalization.2. Purification and Defense StrategyThe defense architecture is inspired by Diffusion Models. The process assumes that adversarial perturbations exist as structured, high-frequency signals that can be "washed out" through controlled noise injection.Defense Pipeline:Forward Noise Injection: Application of Gaussian noise according to a cosine-squared schedule to disrupt adversarial structures.Adaptive Denoising: Use of an adaptive Gaussian filter where the kernel size and standard deviation are dynamically calculated based on the noise-to-signal ratio of the current timestep.Quantization and Compression: Baseline comparisons against JPEG compression (Quality = 15) and 3-bit quantization.Experimental ResultsResNet50 Performance on ImageNetteCategoryOriginal PredictionAttacked PredictionConfidenceImage 0TenchReel100.0%Image 1English Springer SpanielShih Tzu100.0%Image 3ChainsawLorikeet100.0%Defense Efficacy (MNIST)Baseline (No Defense): 0% Accuracy under I-FGSM.Standard Gaussian Blur: 40% Accuracy restoration.Diffusion Purification (Adaptive): 60% Accuracy restoration.Environment and DependenciesThe project is implemented in Python using the PyTorch framework.
